A Simple solution enabling Multiple Values in a Single Excel Field

By Gerard Marshall September 19, 2020


Basic Overview


Introduction: What will be created...

Instead of being limited by one value in the default Data Validation drop down list, you will be creating a single customisable Macro to allow Multiple Selections.

Conclusion: What you will learn...

You learn how to use simple VBA code to create functionality for a Macro within Microsoft Excel.

Requirements: What you need...

  • Microsoft Excel
  • Basic Knowledge of VBA
