Setting up using USB and SSH

By Gerard Marshall September 19, 2020


Introduction: What will be created...

The Goal of this tutorial will be to setup the Raspberry Pi Zero through USB and SSH. Therefore, we will be creating a SD Card Operating system and a Virtual Ethernet Connection.

Conclusion: What you will learn...

You will learn how to Install Raspbian, edit Raspberry PI driver files and setup SSH. This will allow you to easily and efficiently setup your Raspberry PI without the use of any external accessories.

Requirements: What you need...

  • PC
  • Arma 3
  • Steam CMD
  • TDSM
  • PBO Extractor
